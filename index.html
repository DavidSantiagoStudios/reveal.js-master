Archivos planos

Los archivos planos son aquellos que almacenan datos sin formato y estructura definida.

Se caracterizan por ser fáciles de leer y editar con cualquier editor de texto.

Algunos ejemplos de archivos planos son los archivos de texto (.txt), archivos CSV (.csv) y archivos de registro (.log).

Archivos XML

Los archivos XML (Lenguaje de Marcado Extensible) son archivos de texto que contienen información estructurada.

Se utilizan para almacenar y compartir datos de manera independiente del software o hardware utilizado.

Los archivos XML se caracterizan por utilizar etiquetas para definir la estructura y contenido de los datos.

Ejemplos de archivos XML incluyen el formato de documentos Microsoft Office (.docx, .xlsx) y archivos de configuración de software.

Archivos JSON

Los archivos JSON (Notación de Objetos de JavaScript) son archivos de texto que almacenan datos estructurados en formato de objeto.

Se utilizan comúnmente para enviar y recibir datos a través de aplicaciones web.

Los archivos JSON se caracterizan por utilizar llaves para definir la estructura y contenido de los datos.

Ejemplos de archivos JSON incluyen respuestas de API web y archivos de configuración de software.

Archivos Parquet

Parquet es un formato de archivo de datos columnar y comprimido que se utiliza para almacenar grandes conjuntos de datos.

Está diseñado para ser eficiente en términos de almacenamiento y procesamiento de datos.

Parquet es utilizado comúnmente en aplicaciones de big data y análisis de datos.

Se caracteriza por tener un alto rendimiento en consultas y búsquedas en grandes conjuntos de datos, gracias a su estructura columnar y compresión.
